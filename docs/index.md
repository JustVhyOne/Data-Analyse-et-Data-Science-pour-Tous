## 🔭 Aperçu : Data Analyse vs Data Science

Souvent confondus, ces deux mondes sont complémentaires. Voici la différence en mode express :

| 🧐 **Data Analyse**  | 🤖 **Data Science** |
|----------------------|----------------------|
| *"Que s'est-il passé ?"* | *"Que va-t-il se passer ?"* |
| Explorer le **passé** et le **présent** | Prédire le **futur** |
| Outils : Excel, SQL, **Pandas** | Outils : **Scikit-learn**, **TensorFlow** |
| Exemple : "Quel produit a le plus vendu en mars ?" | Exemple : "Quel produit recommander à ce client demain ?" |
| Résultat : Tableaux de bord, graphiques | Résultat : Modèles prédictifs, IA |

Les deux utilisent **Python** comme langage commun. C'est comme avoir un couteau suisse et un laboratoire de recherche dans la même poche. 🎒
